Instruction to run files: If you have got python 3.0 or above installed in your machine, you are able to download and run those files on your computer.

It is possible to double check whether it is installed by running: "py -- version" on the command line. Be sure to enter the same folder where the files are saved.

Once you are there, just run "py" followed by the assingment's name; The question is commented above the python code, for instance:

Assigment 7.1: 

On the command line, run: 
py assingment_7.1.py 
a request returning: "Open your file: " will be shown
Then, type "word.txt" or "romeo.txt" (any of the .txt files available on the folder)
